## 4. 最终产品目标

开发一套达到成熟国际数字资产交易终端水准的高保真模拟交易产品。

持续按模块研究并吸收 Binance、OKX、Bybit、Coinbase Advanced、Kraken Pro 等成熟平台的优点，包括：

- 交易效率。
- 信息层级。
- 下单与持仓管理。
- 风险反馈。
- 订单审计。
- 桌面专业工作区。
- 手机独立交易流程。

不得机械复制品牌、页面结构或把所有功能堆在一个界面。每项功能必须解决明确交易任务。

## 5. 产品边界

- 当前为公开行情驱动的高保真模拟交易平台。
- 不接入真实资金。
- 不提供真实充值或提现。
- 不保存 API Key、Secret、助记词或私钥。
- 不连接真实交易所账户。
- 模拟账户、订单、持仓、成交、账本和设置优先使用本地状态。
- 必须明确标注模拟环境和风险边界，不构成投资建议。

## 6. 当前线上能力

- 专业交易工作区与跨端布局。

- 市价、限价和高级模拟委托。

- 下单执行保护与大额确认。
- 账本派生绩效分析。
- 组合风险中心。

- 图表交易层。
- OCO 高级订单。
