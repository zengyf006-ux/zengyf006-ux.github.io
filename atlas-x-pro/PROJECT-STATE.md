## 2. 产品边界

ATLAS X Pro 是公开行情驱动的高保真模拟交易终端：

- 不接真实资金、充值或提现。
- 不连接真实交易所账户。
- 不收集或保存 API Key、Secret、私钥或助记词。
- 订单、持仓、成交、策略、审计和预警仍为模拟账本或浏览器本地状态。
- 不得把随机演示数据标为实时，不得对外宣称已接入真实交易账户。

## 5. 阶段一已完成能力

### 实时行情与缓存

- `market-data-engine` 统一管理交易对、周期、快照、K 线、订单簿、逐笔成交和实时连接。
- IndexedDB 缓存优先首屏，不再每次空白等待远端响应。
- 切交易对或周期时递增请求代际，取消旧请求并关闭旧连接。
- 旧回包不能覆盖当前周期。
- 连接状态区分 `booting`、`live`、`reconnecting`、`stale`、`offline`。
- 数据过期后退出“实时”状态，不伪造随机价格。

### K 线周期与交互

已支持并验证：

- `1m`、`3m`、`5m`、`15m`、`30m`
- `1h`、`2h`、`4h`、`6h`、`12h`
- `1d`、`1w`

K 线详情包含：时间、周期、开盘、最高、最低、收盘、涨跌额、涨跌幅、振幅、成交量、成交额、EMA10、EMA20、数据源和收盘状态。

交互包含：

- 同一根再次点击取消。
- 关闭按钮、Esc、空白区域、切周期、切交易对和重置均可取消。
- 显示可视区最高价、最低价、最新价线和当前蜡烛倒计时。
- 切周期时保留旧图作非阻塞过渡，不再长时间白屏。

### 专业市场筛选器

- 统一使用公共网关，不维护第二套浏览器直连行情源。
- 支持搜索、成交额、涨跌幅、振幅、点差和价格排序。
- 支持高振幅、低点差、自选同步和最多四市场比较。
- 筛选与比较偏好持久化。
- 网关失败时只使用有效缓存；缓存过期后缺失指标显示 `--`。

### 稳定性修复

- 数据健康中心已迁移到统一行情会话。
- 修复预警表单被实时重绘清空、规则创建竞态、标签页闪回和手机触控尺寸。
- 修复手机审计、持仓卡片、市场入口和追踪止损标记的异步几何竞态。
- 修复市场筛选器搜索在实时重绘后失效。
- 质量层只有在所有手机模块完成启动后才宣布 `ready`。

## 7. 已知待改进项

阶段一解决了行情正确性和 K 线基础，但网站尚未达到最终成品：

- 手机顶部状态、周期和图表工具区仍偏密。
- 手机蜡烛详情卡遮挡图表面积偏大。
- 手机图表、盘口、下单和持仓仍像多个模块拼接，缺少成熟交易所的一体化节奏。
- 订单簿列宽、精度聚合和买卖盘模式仍需强化。
- 下单面板的订单类型、数量/金额切换、费用、滑点和成交均价预估仍需系统重排。
- 桌面低优先级文字偏小，后续应通过信息密度档位解决，不做粗暴全局放大。
