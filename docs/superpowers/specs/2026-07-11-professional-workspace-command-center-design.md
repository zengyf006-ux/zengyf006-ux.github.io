# ATLAS X Pro 专业工作区与命令中心设计

日期：2026-07-11

## 目标

在不改动现有行情、模拟撮合、订单账本和高级退出策略的前提下，为 ATLAS X Pro 增加顶级交易终端常见的高效率工作流：命令面板、工作区模式、交易预设、键盘快捷键和统一安全锁。

本轮不做自由拖拽、任意面板缩放、云同步和真实账户权限系统。原因是这些能力会显著扩大布局回归面，而当前更高价值的是让用户更快找到功能、切换工作状态，并在所有下单入口上获得一致的安全边界。

## 用户体验

### 命令面板

- 点击顶部“搜索市场”或按 `⌘/Ctrl + K` 打开。
- 同一输入框搜索交易对和操作。
- 支持键盘上下选择、Enter 执行、Escape 关闭。
- 操作包括：切换工作区、买入/卖出、切换市价/限价/止盈止损、切换周期、锁定交易。
- 交易对结果显示名称、交易对和当前价格；操作结果显示分类和快捷键。
- 执行任何动作后关闭面板并返回原工作区。

### 工作区模式

工作区状态通过 `document.documentElement.dataset.workspaceMode` 驱动 CSS，不改 DOM 顺序。

1. `standard` 标准：保留当前四列终端。
2. `chart` 图表专注：隐藏市场侧栏、盘口和下单区，主工作区占满；适合分析。
3. `execution` 执行专注：隐藏市场侧栏，保留图表、盘口和下单区；适合下单。
4. `risk` 风险复核：隐藏市场侧栏和盘口，扩大主工作区与下单区，并自动切到持仓视图；适合检查仓位、风险预算和退出策略。

手机端不重排为桌面网格，只保存模式并把对应主视图切到最接近的移动页签。

### 交易预设

内置三套可解释预设，只填参数，不创建订单：

- 保守：限价、Post Only、风险预算 0.5%。
- 均衡：限价、关闭 Post Only、风险预算 1.0%。
- 主动：市价、关闭 Post Only、风险预算 1.5%。

预设不会填入数量，不会自动触发确认弹窗，也不会覆盖止损价和目标价。

### 统一交易安全锁

- 锁定状态存储在 `atlasX.pro.workspace.v1`。
- 锁定后统一拦截普通下单、OCO、追踪止损和分批止盈入口。
- 拦截发生在捕获阶段，早于各模块自己的提交监听器。
- 页面显示持续锁定状态；按钮保留可聚焦能力，用于展示被阻止原因，不使用原生 `disabled`。
- 解锁采用两步确认：第一次请求解锁，3 秒内再次确认才完成。
- 不提供任何“锁定时仍允许自动提交”的旁路。

### 快捷键

仅在焦点不位于输入框、文本域、下拉框或可编辑区域时生效：

- `B` / `S`：买入 / 卖出。
- `M` / `L` / `T`：市价 / 限价 / 止盈止损。
- `Alt + 1` / `Alt + 5` / `Alt + H` / `Alt + 4` / `Alt + D`：1m / 5m / 1H / 4H / 1D。
- `⌘/Ctrl + K`：命令面板。
- `⌘/Ctrl + Shift + L`：锁定；若已锁定则打开解锁确认，不直接解锁。
- 不设置下单快捷键。

## 架构

新增两个独立文件：

- `atlas-x-pro/workspace-command-center.js`
- `atlas-x-pro/workspace-command-center.css`

模块只通过现有 DOM 控件触发点击和输入事件，避免直接修改 `app.js` 内部状态。公开只读/受控接口：

```js
window.AtlasWorkspace = {
  getState(),
  openCommand(),
  closeCommand(),
  setMode(mode),
  applyPreset(id),
  setLocked(locked, options),
};
```

存储结构：

```json
{
  "version": 1,
  "mode": "standard",
  "preset": "balanced",
  "locked": false,
  "updatedAt": 0
}
```

读取失败、字段无效或版本不兼容时回退默认值，不影响核心交易状态。

## 组件

- 顶部布局按钮弹出的工作区面板。
- 桌面居中命令面板；手机底部命令抽屉。
- 工作区模式按钮组。
- 三个预设卡片。
- 安全锁状态块与两步解锁按钮。
- 快捷键说明列表。

所有新增选择器使用 `workspace-` 前缀或 `data-workspace-*`，不复用 `data-side`、`data-symbol` 等核心事件代理字段。

## 数据流

1. 模块启动后读取工作区存储。
2. 应用模式、预设标记和锁定标记。
3. 用户操作通过现有按钮点击或输入事件更新核心界面。
4. 工作区模块只保存自身设置，不写入 `atlasX.pro.v1`。
5. 安全锁在捕获阶段检查提交目标；锁定时停止传播并更新统一状态提示。

## 错误处理

- 存储损坏：忽略并恢复默认。
- 目标控件不存在：动作显示“当前视图不可用”，不抛异常。
- 移动端下单 Sheet 未打开：买卖快捷键先调用现有移动买卖按钮。
- 解锁确认超时：恢复“请求解锁”。
- 命令搜索无结果：显示空状态，不执行最近动作。

## 验收

四种视口：390×844、430×932、1440×900、1920×1080。

必须验证：

1. 命令面板可由按钮和快捷键打开。
2. 搜索 ETH 并执行后切换至 ETH/USDT。
3. 搜索“风险复核”可切换工作区。
4. 工作区模式刷新后恢复。
5. 三个预设准确更新订单类型、Post Only 和风险比例，且不会创建订单。
6. 锁定状态阻止普通订单、OCO、追踪止损和分批止盈。
7. 两步解锁成功，单次点击不能解锁。
8. 输入框内按 B/S/M/L/T 不触发快捷操作。
9. 手机命令抽屉和工作区面板无横向溢出，触控目标不低于 38px。
10. 全部既有交易、风险、图表、OCO、退出策略和市场数据验收零回归。

## 非目标

- 自由拖拽面板。
- 自定义任意网格。
- 云端同步工作区。
- 真实账户 PIN、2FA 或权限角色。
- 键盘直接提交订单。
