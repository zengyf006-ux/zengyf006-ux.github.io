# Legacy Disposition

`atlas-x-pro/` and PR #13 remain reference-only sources for business rules, feature inventory and historical test cases. No old global-script architecture, DOM patching, localStorage business truth, window global state, UI markup or production gateway code is copied into ATLAS X Unified Pro.

The new implementation owns its contracts, domain logic, state machines, persistence adapters, network adapters and UI architecture under `atlas-x-unified/`.
