# ATLAS X Release Candidate 验收记录

测试日期：2026-07-10

## 产品范围

- OKX 公共现货与永续行情，失败时自动使用本地缓存/演示数据。
- 专业 K 线：多周期、EMA、BOLL、VOL、RSI、MACD、单根数据、拖动、缩放、全屏。
- 订单簿、逐笔成交、价格闪烁、盘口深度和资金费率倒计时。
- 现货与永续模拟交易；市价、限价、Post Only、IOC、FOK、Reduce Only。
- 部分成交、撤单、持仓、委托、成交、手续费、保证金、资金流水和账户分析。
- 首页、行情、交易、资产、分析、闪兑、链上与策略页面。
- 全部抽屉支持按钮、遮罩、浏览器返回和 Escape 退出。

## 工程校验

- 独立 `atlas-x/` 工程，不引用 V7—V14 业务脚本。
- 首屏 HTML/CSS 立即呈现，业务包在后台并行加载。
- 基础样式、四套 Release 样式和最终启动器均已完成 Git Blob SHA 比对。
- 压缩业务包首尾分片与本地构建文件一致。
- 本地完整业务包通过 JavaScript 语法检查和多视口浏览器交互测试。

## 产品边界

本项目为公开行情驱动的高保真模拟交易平台，不接入真实资金、真实充值提现或生产级交易所撮合基础设施。
